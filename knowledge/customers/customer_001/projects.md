# Los Andes Cobre — Trommel Screen Panel Retrofit

## Summary

Replaced OEM steel trommel panels with Tega modular polyurethane panels
on both mill-feed trommels in 2025, following repeated steel panel
blinding and short wear life on high-silica ore.

## Installed specification

| Parameter | Value |
|---|---|
| Product | Tega Trommel Panel — modular PU, see `products/trommel_panel.md` |
| Panel size | 305 x 305 mm |
| Panel thickness | 40 mm |
| Aperture (hole) size | 25 mm round |
| Hole pattern | Square pitch, 32 mm centers, staggered rows |
| Open area | ~46% |
| Material | Polyurethane, 90 Shore A, cast |
| Fixing profile | Pin-and-sleeve, top-fix, rubber sleeve |
| Trommel drum diameter | 3.6 m |
| Panels per drum | 184 |
| Install date | March 2025 |

## Why polyurethane over rubber

Ore at this site is dry and abrasive (high silica, low moisture) rather
than impact-heavy, which favors PU's superior abrasion resistance over
rubber's superior impact/cut resistance. See
`rules/design_rules.md` §"Material selection" for the general rule.

## Status

Active, in service since March 2025. First scheduled wear inspection
completed January 2026 — see `reports/wear_report_2026_01.md`.
